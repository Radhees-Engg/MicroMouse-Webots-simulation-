# MicroMouse-Webots-simulation-
 
Autonomous Micromouse maze
exploration using a DFS based
algorithm with coordinate mapping 
and stack based backtracking, 
implemented in the Webots
simulation environment.

## Key Features

- DFSbased maze exploration
- Grid based environment mapping
- Dynamic wall detection using sensors
- Stack based backtracking for dead end recovery
- Implemented and tested in the Webots robotics simulator

## Future Work

- Implement Flood-Fill or A* for optimal path planning
- Add multi-robot collaborative exploration
- Extend to real Micromouse hardware platforms


0 - path, 8 - wall, 1 - visited,
3 - Dead end (No Unexplored Cells nearby).
